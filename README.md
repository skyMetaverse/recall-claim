使用ethers.js在Base主网上领取代币的脚本。

## 安装

确保已安装依赖：

```bash
npm install
```

## 配置

1. 复制环境配置文件：
```bash
cp .env.example .env
```

2. 编辑`.env`文件，填入你的钱包私钥：
```
PRIVATE_KEY=你的私钥（不要包含0x前缀）
```

## 使用方法

运行领取脚本：

```bash
node claim.js
```

或者设置环境变量后运行：

```bash
PRIVATE_KEY=你的私钥 node claim.js
```

## 脚本功能

- ✅ 连接Base主网
- ✅ 自动估算Gas费用
- ✅ 发送领取交易
- ✅ 等待交易确认
- ✅ 详细的日志输出
- ✅ 错误处理和提示

## 注意事项

1. **安全警告**：请妥善保管你的私钥，不要提交到版本控制系统
2. **Gas费用**：确保钱包有足够的ETH支付交易费用
3. **网络**：脚本连接Base主网，请确认网络环境正常
4. **领取限制**：每个地址通常只能领取一次
